# Platform Onboarding: Getting Started

## Target User
Developers who want to solve project problems with AI-assisted workflows and reusable context.

## First Session Flow
1. Login to the platform
2. Configure MCP connection required for your workflow
3. Open/create your first post in Studio
4. Save draft and validate rendering/publish path

## User Readiness Checklist
- willing to learn new workflow patterns
- can define problems clearly
- can make explicit decisions with constraints
