# 03-editor-ui

Create lightweight markdown editor with toolbox and image upload shortcode insertion.

## Inputs
- `src/components/editor/MarkdownEditor.tsx`
- `src/lib/editor/upload.ts`

- textarea + toolbar
- markdown wrapping/prefixing/insertion behavior
- image upload + shortcode insertion

## Done Criteria
- title/body typing preserved
- toolbox actions mutate markdown text predictably
